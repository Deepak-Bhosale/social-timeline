# social-timeline

## Setup

To set up and run the web application, follow these steps:

1. Clone the repo using below link

   ```bash
    git clone git@github.com:Deepak-Bhosale/social-timeline.git
   ```

2. Navigate to the react-setup Directory:
   ```bash
    cd social-timeline
   ```
3. Install the project dependencies:

   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```
